# fincal-hackathon

A Next.js app built for Technex'26 FinCal Innovation Hackathon (HDFC Mutual Fund). 

## Features implemented
- SIP Calculator with exact formula
- SWP Calculator with exact formula
- Dual input mode (range slider + numeric field)
- Live recalculation (reactive inputs)
- Chart-based summary (recharts)
- Accessibility (WCAG 2.1 (A/AA))
- Full disclaimer text required by contest
- Input validation with screen-reader alerts
- Unit tests for calculations

## Local setup
```bash
npm install
npm run dev
```
Visit http://localhost:3000, then `/sip` or `/swp`.

## Deploy
Connect to Vercel or run on any static Node platform using standard Next.js build:
```bash
npm run build
npm run start
```
